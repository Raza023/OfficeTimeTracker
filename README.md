# 🚀 Office Suite: Ultimate Time Tracker

[Office Suite](https://raza023.github.io/OfficeTimeTracker/)

A sleek, modern, and high-performance **Office Time Tracking** application designed with a premium glassmorphism aesthetic. Whether you want to track your hours live as you work or manually input your schedule, Office Suite makes it effortless and beautiful.

---

## ✨ Key Features

- **⏱ Live Tracker**: Real-time office hour tracking with start/stop functionality.
- **☕ Smart Breaks**: Easily pause your session for breaks and resume with a single click.
- **📝 Manual Mode**: Log your hours manually by entering start, break, and end times.
- **🎨 Glassmorphism UI**: A stunning, semi-transparent interface with vibrant gradients and smooth animations.
- **📱 Responsive Design**: Optimized for both desktop and mobile devices.
- **🔔 Interactive Toasts**: Beautiful notifications for session summaries.

---

## 🛠 Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Advanced animations, glassmorphism, and responsive layouts.
- **JavaScript**: Lightweight, zero-dependency logic for time calculations.

---

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to start tracking your productivity.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/OfficeTimeTracker.git
   ```
2. **Open index.html:**
   Open the file directly in your browser or serve it using a local development server.
